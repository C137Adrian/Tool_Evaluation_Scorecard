# 📘 Choosing Your Vibe Coding Tools — Mi Proyecto

Este repositorio contiene mi análisis y selección de herramientas para Vibe Coding, siguiendo el proyecto de 4Geeks Academy.  
El objetivo es identificar qué herramientas se ajustan mejor a mis necesidades reales como desarrollador.

---

## 🌱 1. Tipo de proyecto

Estoy trabajando en:

➡️ **Proyectos reales orientados a backend/data engineering**, con necesidad de:
- Control total del código  
- Mantenibilidad  
- Calidad  
- Escalabilidad  
- Integración con mi stack (Python, APIs, SQL, cloud)

---

## 🎯 2. Tareas donde usaré herramientas de Vibe Coding

- Generación de código backend  
- Refactorización  
- Documentación  
- Automatización  
- UI rápida cuando sea necesario  
- Soporte para arquitectura y planificación  

---

## ⛓️ 3. Mis restricciones

- **Presupuesto limitado** (prefiero herramientas gratuitas o con buen free tier)  
- **Tiempo limitado** (quiero herramientas que aceleren, no que compliquen)  
- **Stack existente**: Python, VS Code, GitHub, cloud  
- **Evitar plataformas cerradas** que limiten el control del código
